## 一、布局思路
整体采用 Column 垂直布局 实现名片页面，从上到下依次排列：
展示信息：头像、姓名、身份、电话、社交账号、邮箱
- 顶部展示个人头像图片；
- 中间展示姓名、英文标题（美食家定位）；
- 底部通过 Row 横向布局 实现三组联系方式（图标 + 文字）；
- 使用 Modifier 控制尺寸、间距、背景色，整体居中对齐，风格简洁优雅。

## 二、使用的 Composable
Column、Row、Surface、Image、Text、Icon、Spacer
- 嵌套关系   
最外层：MainActivity → Surface  
内层：Surface → BusinessCard → Column（根布局）
Column 内依次嵌套：
Image → Spacer → Text → Spacer → Text → Spacer
底部：Column 中嵌套 3 个 ContactItem
每个 ContactItem 内部：Row 嵌套 Icon + Spacer + Text


## 三、遇到的问题与解决方式
- 图标 / 类型相关报错、
- 问题：ImageVector 类型导入失败、函数参数类型不匹配，代码大量标红。
- 解决：放弃复杂类型声明，改用独立图标函数写法，避开类型兼容问题，保证代码可运行。
导包冗余 / 缺失
- 问题：自定义头像图片无法正常显示，系统提示资源加载失败。
- 解决：将图片放入正确的资源文件夹，通过资源 ID 正确引用，确保图片路径与名称无误。
界面样式调整
- 问题：Compose 预览界面空白，无法实时查看界面效果。
- 解决：正确添加预览注解，刷新项目并重新构建，使预览功能正常工作。
## 四、最终效果
完成了一个个性化名片，包含头像、姓名、身份描写、联系方式，界面美观、代码简洁、可正常运行预览。
